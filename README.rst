UNICEF Attachments
==================

Django package that handles file attachments for models


Installation
------------

    pip install unicef-attachments


Setup
-----

Add ``unicef_attachments`` to ``INSTALLED_APPS`` in settings
.. code-block:: python

    INSTALLED_APPS = (
      ...
      'unicef_attachments',

    )
